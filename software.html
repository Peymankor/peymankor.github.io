Software

All publicly available software that I maintain is available on my Github.

corncob: Count Regression for Correlated Observations with the Beta-binomial

An R software package an individual taxon regression model that uses abundance tables and sample data. It models differential abundance and differential variability, and addresses different sequencing depths, excessive zeros, overdispersion, within-taxon correlation, and hypothesis testing with categorical and continuous covariates.
Availablility: Github
Reference: Available here.


DivNet: Diversity Estimation in Networked Communities

An R software package for estimating community diversity in microbial communities where taxa interact via an ecological network.
Availablility: Github
Reference: Available here.


SUMMER: Spatio-temporal Under-five Mortality Model Estimation in R

An R software package providing a user-friendly implementation of methods for estimating, projecting, and plotting spatio-temporal under-five mortality rates.
Availablility: CRAN and Github
Reference: Available here.


WSC

An R software package intended for personal use by myself and friends. You may find it useful if you play DnD, want to split checks fairly, or want to be guided through the process of updating R in OSX without losing your packages.
Availablility: Github
